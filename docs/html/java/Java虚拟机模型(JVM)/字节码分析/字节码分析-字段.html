package java.bytecode;

/**
 * 字段示例 - 演示各种类型的字段定义
 */
public class FieldExample {

    // 静态字段
    public static final int STATIC_FINAL_INT = 100;
    private static String staticField = "static";

    // 实例字段 - 基本类型
    private int intField;
    private long longField;
    private double doubleField;
    private boolean booleanField;
    private char charField;

    // 实例字段 - 对象类型
    private String stringField;

    // 实例字段 - 数组
    private int[] intArray;
    private String[] stringArray;

    // final字段
    private final String finalField = "final";

    public FieldExample() {
        this.intField = 0;
    }

    public int getIntField() {
        return intField;
    }

    public void setIntField(int intField) {
        this.intField = intField;
    }
}

详细说明

字段描述符

字节码中使用描述符来表示字段类型：

  • I - int类型
  • J - long类型
  • D - double类型
  • Z - boolean类型
  • C - char类型
  • Ljava/lang/String; - String对象类型（L开头，分号结尾）
  • [I - int数组
  • [Ljava/lang/String; - String数组

字段访问标志

  • ACC_PUBLIC - public访问权限
  • ACC_PRIVATE - private访问权限
  • ACC_STATIC - static静态字段
  • ACC_FINAL - final不可变字段

ConstantValue属性

对于编译时常量（如public static final int STATIC_FINAL_INT = 100和private final String finalField = "final"），编译器会将常量值存储在ConstantValue属性中，而不是在构造方法或静态初始化块中赋值。

静态初始化块

<clinit>方法是类初始化方法，用于初始化静态字段。在本例中，staticField的初始化在<clinit>方法中完成：

字段访问指令

  • getfield - 获取实例字段值
  • putfield - 设置实例字段值
  • getstatic - 获取静态字段值
  • putstatic - 设置静态字段值
