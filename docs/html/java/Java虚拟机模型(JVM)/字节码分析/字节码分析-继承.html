字节码分析-继承

源码

package java.bytecode;

import java.bytecode.BaseClass;

/**
 * 继承示例 - 演示继承、方法重写、super调用
 */
public class InheritanceExample extends BaseClass {

    private String childField = "Child";

    public InheritanceExample() {
        super();
    }

    public InheritanceExample(String message) {
        super(message);
    }

    // 重写父类方法
    @Override
    public void baseMethod() {
        super.baseMethod();
        System.out.println("InheritanceExample.baseMethod()");
    }

    // 调用父类protected方法
    public void callProtectedMethod() {
        protectedMethod();
    }

    public String getChildField() {
        return childField;
    }
}

详细说明

继承关系

在字节码中，继承关系通过类的定义和父类索引表示：

public class java.bytecode.InheritanceExample extends java.bytecode.BaseClass

super调用

使用invokespecial指令调用父类方法，这是super关键字在字节码中的实现：

public InheritanceExample();
  Code:
     0: aload_0
     1: invokespecial #1  // Method java/bytecode/BaseClass."<init>":()V

invokespecial用于：
- 调用父类构造方法（super()）
- 调用父类方法（super.baseMethod()）
- 调用私有方法

方法重写

子类重写父类方法时，使用invokevirtual调用子类方法，实现多态：

public void baseMethod();
  Code:
     0: aload_0
     1: invokespecial #18  // Method java/bytecode/BaseClass.baseMethod:()V
     4: getstatic     #21  // Field java/lang/System.out:Ljava/io/PrintStream;
     7: ldc           #27  // String InheritanceExample.baseMethod()
     9: invokevirtual #29  // Method java/io/PrintStream.println:(Ljava/lang/String;)V
    12: return

protected方法调用

子类调用父类的protected方法时，使用invokevirtual指令：

public void callProtectedMethod();
  Code:
     0: aload_0
     1: invokevirtual #34  // Method protectedMethod:()V
     4: return

方法调用指令的区别

  • invokespecial - 用于调用父类方法、构造方法、私有方法，不支持多态
  • invokevirtual - 用于调用实例方法，支持多态（虚方法调用）
  • invokestatic - 用于调用静态方法
  • invokeinterface - 用于调用接口方法

虚方法表（vtable）

JVM使用虚方法表实现多态。当调用invokevirtual时，JVM会根据对象的实际类型查找虚方法表，找到正确的方法实现。
