## 类加载器

### 双亲委派机制
- 类加载器接收到加载请求，先检查该类是否已被加载（缓存），若已加载直接返回Class对象；
- 若未加载，委托给父加载器处理；
- 父加载器重复步骤 1-2，直到启动类加载器；
- 启动类加载器检查是否能加载：能则加载并返回，不能则向下传递给扩展类加载器；
- 扩展类加载器同理：能则加载，不能则传递给应用程序类加载器；
- 应用程序类加载器仍无法加载，则由自定义类加载器（若有）调用findClass()加载；
- 若所有加载器都无法加载，抛出ClassNotFoundException。
